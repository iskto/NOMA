##簡介

主要功能有旅遊行程規劃、分享、購買以及導遊等功能。
Noma提供使用者可以完整自行規劃旅遊的行程細節，行程規畫完後可以將行程儲存在手機中，供使用者之後可以自行瀏覽，若覺得行程規畫得非常順利且有趣，可以更進一步將此行程分享到我們所建立的網路分享平台上供其他使用者瀏覽。
而在旅行過程中可以將沿途景色或美食等照片依日期一併記錄在此次中，使行程記錄更加豐富，讓使用者可以隨時回憶此次旅遊，重溫旅程的美好。
